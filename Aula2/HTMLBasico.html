Programação WEB I

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Basicamente, ela define a estrutura e o conteúdo de um site, permitindo que o navegador exiba textos, imagens, vídeos e outros elementos de forma organizada. Não é uma linguagem de programação, mas sim uma linguagem de marcação que usa tags para definir a estrutura do conteúdo.
